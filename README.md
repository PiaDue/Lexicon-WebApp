# Lexicon-WebApp
Project was part of a php course 

## Features
- lexicon overview
- search function
- detail page for each term
- admin mode
  - login and logout
  - create new term-entry
  - edit term-entry
  - delte term-entry
  
## Planned Features
- moving admin login data to MySql DB
- register for new admins
